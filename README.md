# IOM EPPI Evidence Maps

Interactive evidence maps of IOM evaluation reports (as available on the [IOM Evaluation Portal](https://evaluation.iom.int/evaluation-search-pdf)), produced using [EPPI Mapper](https://eppi.ioe.ac.uk/cms/er4/eppi-mapper) and following the approach described at [fr.anckalbi.net/iomeval](https://fr.anckalbi.net/iomeval).

The maps organise evaluation evidence around the 23 objectives of the [Global Compact for Safe, Orderly and Regular Migration (GCM)](https://www.iom.int/global-compact-migration) — UN General Assembly Resolution [A/RES/73/195](https://digitallibrary.un.org/record/1660537), adopted on 19 December 2018.

## Maps

| Map | Description |
|-----|-------------|
| [Regions × GCM Objectives](http://fr.anckalbi.net/iom-eppi/eppi-evmap-regions-gcms.html) | Evaluation evidence mapped by world region against GCM objectives |
| [Years × GCM Objectives](http://fr.anckalbi.net/iom-eppi/eppi-evmap-years-gcms.html) | Evaluation evidence mapped by publication year against GCM objectives |
